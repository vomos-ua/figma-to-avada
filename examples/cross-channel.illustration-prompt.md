# Cross-channel Remarketing — Illustration

**Aspect ratio:** 4:3 landscape (~1453×1093 px)
**Placement:** Right column of Cross-channel section (alongside 2x2 step cards)
**Source:** User-provided reference image (already exists, no regeneration needed)

If regeneration ever needed, this is the ChatGPT-Image prompt:

```
Create a clean, modern isometric flat UI illustration for a B2B marketing landing page. The image visualizes "Cross-channel remarketing": THREE ad platforms (Facebook, Google, LinkedIn) each show a different ad to the same prospect, all converging into a single landing page that converts to one lead.

STYLE:
- Clean, minimalist flat-3D / isometric vector style
- Pure white background (#FAFBFC) with very subtle blue ambient glow in upper-right corner
- White card panels with 16px rounded corners and very soft drop shadows (rgba(13,17,38,0.06))
- All connecting arrows are short dotted/dashed lines with arrowheads, in different pastel hues: light purple (#A8B5F0), light blue (#7BB8F5), light teal (#7DD3C7)
- Brand accent color: deep blue #2134EA used sparingly on key icons, badges, and CTAs

COMPOSITION (4:3 landscape, ~1453×1093 px):

LEFT side (1/3 of canvas) — THREE ad cards stacked vertically:
- TOP card: Facebook-blue square logo with thumbs-up icon + photorealistic mini chair/furniture image + blue progress bar at bottom
- MIDDLE card: Google multi-color "G" logo + photorealistic mini sneaker/shoe image + blue progress bar at bottom
- BOTTOM card: LinkedIn-blue square logo + photorealistic mini head shot of confident business professional in glasses + blue progress bar at bottom

CENTER/RIGHT (2/3 of canvas):
- Browser window mockup: rounded corners, three traffic-light dots (red/yellow/green) in upper-left, mock URL bar, hero image area showing stylized snowy mountain landscape, three placeholder text lines underneath, one prominent blue CTA button
- Three small feature icons in white cards below browser (icons: network nodes, target with crosshair, shield-with-checkmark — all in deep blue #2134EA)
- Below browser: single user profile circle (white card, blue user silhouette icon, small blue checkmark badge in lower-right)
- Below user circle: "Conversion" card with purple-to-violet gradient circle containing white trophy/award

FLOW:
- Three dotted arrows (each in different pastel color: purple/blue/teal) from each ad card on the left → converging into the browser mockup
- One dotted arrow from browser down to user circle
- One dotted arrow from user circle down to trophy card

Tone: professional, B2B SaaS, clean, modern, optimistic, slightly futuristic.
```
